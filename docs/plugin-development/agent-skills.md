# Agent Skills

Agent Skills enable contextual discovery of plugin capabilities.

## SKILL.md Format

Agent Skills use a SKILL.md file to document plugin capabilities for AI agents.

## Example

```markdown
---
name: My Skill
description: Does something useful
---

## Context Keywords

- keyword1
- keyword2

## Tools

- tool_name - Description

## Instructions

How to use this skill effectively.
```

See `docs/examples/example-git-skill/SKILL.md` for a complete example.
