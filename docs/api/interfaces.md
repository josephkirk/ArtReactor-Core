# Core Interfaces

## Plugin Interfaces

See [Plugin System](../architecture/plugin-system.md) for complete interface documentation.

### Plugin Base Classes

- `Plugin` - Base interface
- `CorePlugin` - General purpose plugins
- `RouterPlugin` - API endpoint plugins
- `AppPlugin` - External app control
- `AgentPlugin` - Custom agents
- `ModelPlugin` - AI model providers
- `UiPlugin` - Web interfaces
